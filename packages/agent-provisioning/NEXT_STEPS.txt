====================== KEY IMPLEMENTATION POINTS ========================

Trial Model:
  → User signs up email: john@example.com + wallet: ABC123 
  → Gets 1 trial (tied to this combo)
  → Can't signup again with same email (even different wallet)
  → Can't signup again with same wallet (even different email)
  → One trial per email+wallet = one trial per user

Payment Model:
  → User provides their own API key (e.g., OpenAI key)
  → HFSP charges $9/month for provisioning/hosting
  → User pays OpenAI separately for actual token usage
  → Simpler billing (fixed rate, not metered)

Scope for 3 Days:
  → Auth system (email + Phantom)
  → Paywall (trial → Pro)
  → UI Wizard (signup + create agent)
  → Mock provisioning (returns success after 5 sec)
  → NOT real Docker provisioning (that's Week 2)
